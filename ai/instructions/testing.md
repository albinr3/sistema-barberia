# Instrucciones De Testing

## Proposito

Guia inicial para pruebas futuras del monorepo.

## Reglas

- Mantener pruebas separadas por componente tecnico.
- Cubrir reglas de negocio confirmadas cuando sean implementadas.
- No crear suites de prueba antes de existir el proyecto correspondiente.
